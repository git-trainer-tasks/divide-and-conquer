# log-filter

`log-filter` — небольшая утилита командной строки для поиска записей в текстовых логах.

## Сборка

Проект состоит из нескольких простых C++-файлов и не требует отдельной системы сборки:

```bash
g++ main.cpp filter.cpp -o log-filter
```

## Использование

Вывести все записи с нужным уровнем логирования:

```bash
./log-filter <level> <file>
```

Вывести только количество подходящих записей:

```bash
./log-filter --count <level> <file>
```

Сравнение уровней не зависит от регистра. Например, уровень `error` подходит для строк,
которые начинаются с `[ERROR]`, `[Error]` и `[error]`.

## Примеры

```bash
./log-filter error app.log
./log-filter --count warn app.log
```

Входной файл должен содержать одну запись лога на строку. Записи, которые не начинаются
с запрошенного уровня, пропускаются.
